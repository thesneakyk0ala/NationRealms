# Roadmap

## Phase 1: Foundation Shell

Create the monorepo, shared types, Prisma schema, demo seed data, Fastify API, Socket.IO scaffold, and React prototype UI.

## Phase 2: Nation Creation and Profile Customization

Completed for the foundation prototype: a multi-step founding wizard, shared creation constants, stat preview, starter package generation, public profile page, and backend creation service. Future work should add authenticated ownership, editing existing profiles, richer flags, and moderation-ready public profile fields.

## Phase 3: Event Engine

Completed for the foundation prototype: authored event templates, eligibility rules, weighted event generation, turn advancement, event choice resolution, stat effects, light agent/map/military hooks, generated posts, and event history.

## Phase 4: News/Media Roleplay Feed

Expand posts into speeches, official updates, image/video embeds, public profiles, timelines, moderation hooks, and nation-facing feeds.

## Phase 5: 2D World Map Expansion

Add terrain, regions, ownership, resource distribution, map generation tools, location upgrades, and richer map UI.

## Phase 6: Character Agents and XP Systems

Add assignments, XP rules, leveling, traits, injuries, loyalty changes, speeches, governance actions, and location improvement actions.

## Phase 7: Military Positioning and Conflict Rules

Add movement costs, supply, unit roles, defensive posture, contested locations, limited engagements, and conflict resolution rules.

## Phase 8: Diplomacy and Roleplay Between Nations

Add embassies, treaties, public statements, private diplomatic notes, sanctions, trade agreements, and multiplayer event hooks.

## Phase 9: AI-Driven Events and Character Behavior

Use AI to help draft events, character reactions, public briefings, and emergent complications while keeping game-state changes server-authoritative.

## Phase 10: Life-Sim Layer for Player Avatar

Add the player representative character, personal schedule, relationships, appearances, speeches, travel, and late-game life-sim mechanics.
